# Safe Keystroke Logger (Tkinter)

This is a small educational Tkinter app that logs keys typed **only inside the app window**.
Use it for learning or accessibility/typing research with consent.

**Files**
- `app.py` — the Tkinter app
- `.gitignore` — ignore logs and envs

**Run**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt  # no deps required for this simple app
python app.py
